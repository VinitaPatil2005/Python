# Python Learning Workspace

This workspace is organized into small topic-based examples and notes for learning Python step by step.

## Structure

- `01_Basics` - first programs, variables, operators, and input
- `02_Strings` - string operations and formatting
- `03_Collections` - lists, tuples, sets, and dictionaries
- `04_Control_Statements` - branching and loops
- `05_Functions` - functions, recursion, scope, and lambda
- `06_OOP` - object-oriented programming examples
- `07_File_Handling` - reading, writing, and deleting files
- `08_Modules` - modules and packages
- `09_Exception_Handling` - error handling examples
- `10_Advanced` - regex, JSON, datetime, and math module examples
- `Notes` - markdown notes and study material
